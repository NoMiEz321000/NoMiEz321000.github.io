Introduction

This project analyzes flight data to detect potential aircraft conflicts. A conflict occurs when two aircraft come closer than 3 nautical miles horizontally and 1000 feet vertically. The system reads aircraft position data (latitude, longitude, altitude, time) and identifies when such conflicts happen.

How To Setup the System

  1. Upload CSV file containing aircraft data (time, lat, lon, altitude, callsign).
  2. The program calculates horizontal distance using the Haversine formula (2D spherical distance).
  3. It calculates vertical separation in feet.
  4. If horizontal < 3 NM and vertical < 1000 ft â†’ conflict detected.
  5. Results are shown in a summary.

Technical Details

  • Programming Language: Python
  • Libraries: pandas, math, tqdm
  • Algorithm: Haversine distance + altitude difference check
